# StayDry Command™ — Deployable Demo Package

This is a Vite + React demo package for the StayDry Command™ concept.

## What is included

- iPhone-friendly responsive UI
- Assessment dashboard
- Pump analytics simulation
- WBS field workflow
- ROI/callback savings calculator
- Smart sump + battery upsell model
- Advanced AI/IoT future feature simulation
- Client proof report
- Fix-It-Once Score
- Built-in demo tests
- PWA manifest for Add to Home Screen on iPhone

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in Terminal.

## Build for production

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Create a free Vercel account.
2. Upload this folder to GitHub or import it directly with the Vercel CLI.
3. Framework: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`

Vercel's current docs confirm Vite is supported and Vercel creates a unique preview URL for deployments.

## Deploy to Netlify

1. Create a free Netlify account.
2. Drag and drop this folder if using manual deploy after building, or connect a GitHub repo.
3. Build command: `npm run build`
4. Publish directory: `dist`

Netlify's current docs identify `npm run build` and `dist` as standard Vite build settings.

## iPhone Demo

After deployment:
1. Open the live URL in Safari on iPhone.
2. Tap Share.
3. Tap Add to Home Screen.
4. Launch it like an app.
